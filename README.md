# PSDwell Repository

This repository is primarily a container for the **`dwellsecure/`** project.  
**If you are looking for the main application and documentation, please go to `dwellsecure/`.**

---

## Where is the important stuff?

- **Main app & docs**: All core source code, backend, and detailed documentation live in **`dwellsecure/`**.  
- **Start here**: See `dwellsecure/README.md` for:
  - Project overview and structure  
  - How to run the backend and frontend  
  - What to expect after running the app  
  - User manual (`docs/USER_MANUAL.md` and `docs/USER_MANUAL.pdf`)  

In short: **treat `dwellsecure/` as the real project root.**

---

## Cloning this repo

```bash
git clone https://github.com/SyHilichurl/psdwell.git

```

From there, follow the instructions in `dwellsecure/README.md`.
