# Dwell Secure – Final Codebase

This repository contains the **final codebase for the Dwell Secure app** for the course GitHub Classroom assignment.

---

## Where is the important code?

- **Main app & docs**: All core source code, backend, and detailed documentation live in the **`dwellsecure/`** directory.  
- **Start here**: Open `dwellsecure/README.md` for:
  - Project overview and structure  
  - How to run the backend and frontend  
  - What to expect after running the app  
  - User manual information (`docs/USER_MANUAL.md` and `docs/USER_MANUAL.pdf`)  

In short: **treat `dwellsecure/` as the real project root.**

---

## How to run

1. Clone this assignment repository:

   ```bash
   git clone https://github.com/GIX-Luyao/final-codebase-dwellsecure.git
   cd final-codebase-dwellsecure/dwellsecure
   ```

2. Follow the instructions in `dwellsecure/README.md` to start the backend and frontend.
